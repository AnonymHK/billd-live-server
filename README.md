# 简介

billd 直播间后端

## 安装依赖

```bash
pnpm install
```

## 启动项目

```bash
# pnpm run dev，运行在4300端口
pnpm run dev
# 或者pnpm run dev:beta，运行在4300端口
pnpm run dev:beta
# 或者pnpm run dev:prod，运行在4200端口
pnpm run dev:prod
```

# 本地环境

- 操作系统：mac os 13.3.1（macbookpro 2020 m1）
- node 版本：16.16.0

# 服务器环境

- node 版本：14.19.0
- redis 版本：5.0.3
- mysql 版本：8.0.26
